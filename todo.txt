"MODES":

- anywhere
    x `s` => status
    x `l` => log
    x `b` => branches
    x `t` => tags

- status (changed files, new files, etc) selectable!!
    x `c` => commit selected if there are selected files, else commit all
    x `U` => revert selected if there are selected files, else revert all
    x `d` => diff selected if there are selected files, else diff all
    - `L` => resolve taking local
    - `O` => resolve taking other

x log
    x `c` => checkout selected revision
    x `d` => show revision details

    x `f` => fetch
    x `p` => pull
    x `P` => push

x revision details
    x `d` => diff selected files if there are selected files, else diff all

x branches
    x `c` => checkout selected branch
    x `n` => new branch
    x `D` => delete branch

x tags
    x `c` => checkout selected tag
    x `n` => new tag
    x `D` => delete tag
