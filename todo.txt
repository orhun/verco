"MODES":

- anywhere
    - `s` => status
    - `l` => log
    - `r` => conflicts
    - `b` => branches
    - `t` => tags

- status (changed files, new files, etc) selectable!!
    - `c` => commit selected if there are selected files, else commit all
    - `U` => revert selected if there are selected files, else revert all
    - `d` => diff selected if there are selected files, else diff all

- log
    - `u` => checkout selected revision
    - `U` => revert to selected revision (??)
    - `d` => show revision details

    - `f` => fetch
    - `p` => pull
    - `P` => push

- revision details
    - `d` => diff selected files if there are selected files, else diff all

- branches
    - `u` => checkout selected branch
    - `n` => new branch
    - `d` => delete branch

- tags
    - `u` => checkout selected tag
    - `n` => new tag
    - `d` => delete tag

- conflicts
    - `o` => resolve taking other
    - `l` => resolve taking local
